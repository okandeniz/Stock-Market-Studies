### ğŸ“Œ Project Overview

This repository is a comprehensive financial data science project that includes:

- **Algorithmic Trading Strategies:**  
  Backtested trading strategies built with technical indicators and statistical methods.

- **Trading Strategy Development:**  
  Over a one-year period, selecting the six BIST100 stocks with the best weekly returns one year prior, selling the three worst performers of these six stocks the following week, replacing these three stocks with the three stocks that delivered the best returns in that week, and continuing this process to the present day. Backtesting is used to test the strategy's reliability, comparing the one-year cumulative returns of the strategy and the BIST100 index, and observing the difference in returns.

- **Machine Learning Models for BIST100 Forecasting with Macroeconomic Variables:**  
  ML models built using macro indicators such as interest rates, inflation and money supply.

- **S&P500 Index Forecasting with LSTM:**  
  Deep learning LSTM models to predict future index movements.

- **Stock Price Forecasting with SARIMA:**  
  Time series modeling using SARIMA to predict future stock prices.

- **Sector-Based Ratio Analysis via Web Scraping:**  
  Balance sheet data is scraped for all companies in a sector, followed by  
  comparative valuation using **EV/EBITDA, P/E, P/B**, and other key ratios.

- **Financial Statement Analysis & Valuation in Excel:**  
  Valuation studies of companies based on income statements, balance sheets and multiplier analysis.
